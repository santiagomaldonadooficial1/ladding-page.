# ladding-page

Es una pagina que se especializa solamente en atraer clientes para el negocio.
Esta pagina de ***obsidian coffee*** es prueba de lo que  es una ladding page real.

## tecnologias
Utilice ***CSS y HTML*** para esta pagina.

## caracteristicas
esta ladding page se caracteriza por tener animaciones, pero no tan saturadas para
cargar la pagina de demacidas cosas, tanbien cuenta con links que llevan directo
a secciones sin tener que bajar. Esta adaptada para todo tipo de dispositivo.
